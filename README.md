# 🌟 我的智能工具箱 (My Smart Toolbox)

一个完全自定义、支持云同步的个人工具箱，帮助您集中管理常用网站和在线工具，提高工作和学习效率。

> **作者**： 刘怀庆 (AI辅助开发)  
> **版本**：v5.0 (云存储版)  
> **最后更新**：2026年1月29日

## ✨ 主要特性

| 功能模块 | 描述 |
|---------|------|
| **🔐 用户系统** | 注册登录、头像上传、云存储支持 |
| **☁️ 云端同步** | 多设备数据同步，永不丢失配置 |
| **📦 模块管理** | 创建、编辑、拖拽排序自定义模块 |
| **🔧 工具管理** | 添加、编辑、删除工具（支持图标上传） |
| **🎨 主题定制** | 个性化页面背景和介绍栏样式 |
| **🔍 智能搜索** | 站内工具搜索 + 多引擎网页搜索 |
| **📱 响应式设计** | 适配桌面端和移动端 |

## 🚀 快速开始

### 在线体验
1. 直接访问部署的网页
2. 点击左上角头像进行注册/登录
3. 开始创建您的个性化工具箱

### 本地运行
```bash
# 1. 克隆仓库
git clone https://github.com/yourusername/smart-toolbox.git

# 2. 打开项目文件夹
cd smart-toolbox

# 3. 打开主文件
open index.html  # Mac
# 或
start index.html # Windows
```

### Supabase 配置（用于云同步）
1. 前往 [Supabase官网](https://supabase.com) 创建新项目
2. 获取项目URL和anon key
3. 在代码中替换配置：
```javascript
const SUPABASE_URL = '你的项目URL';
const SUPABASE_ANON_KEY = '你的anon key';
```

## 🛠️ 核心功能详解

### 1. 用户系统
- **注册登录**：邮箱验证，安全可靠
- **头像管理**：支持上传自定义头像
- **云存储**：所有配置自动同步到云端
- **密码管理**：可在线更改密码

### 2. 工具箱管理
- **模块创建**：自定义模块名称、图标、颜色、布局
- **工具添加**：添加工具链接、图标、描述
- **拖拽排序**：在编辑模式下自由拖拽模块和工具
- **导入导出**：支持配置的备份和迁移

### 3. 个性化设置
- **页面背景**：纯色背景或自定义图片背景
- **介绍栏**：自定义标题、内容和背景
- **搜索设置**：选择默认搜索引擎

### 4. 智能搜索
- **站内搜索**：快速查找已添加的工具
- **网页搜索**：支持百度、Google、Bing、知乎、GitHub
- **搜索结果**：实时显示匹配项

## 💾 数据存储

### 云端存储（Supabase）
- **用户配置**：存储在 `user_configs` 表
- **用户头像**：存储在 `user-avatars` 存储桶
- **数据安全**：行级安全策略保护用户隐私

### 本地存储（LocalStorage）
- 未登录时的临时数据存储
- 提升页面加载速度

## 📈 部署与扩展

### 简单部署
1. 将 `index.html` 上传到任何静态网站托管服务
2. 配置 Supabase 项目白名单（如果需要）
3. 分享链接即可使用

### 推荐托管平台
- **GitHub Pages**：免费、简单
- **Vercel**：自动部署、全球CDN
- **Netlify**：类似Vercel，功能丰富

## 📋 使用提示

1. **初次使用**：建议先注册账户以启用云同步功能
2. **编辑模式**：进入编辑模式后可拖拽排序模块和工具
3. **快速搜索**：按Enter键可直接使用当前搜索引擎搜索
4. **数据备份**：定期导出配置以防万一
5. **图片上传**：建议使用小于1MB的图片以获得最佳性能
